# task-management-dashboard

## Description
Task Management Dashboard is a responsive web application built with HTML, CSS, and JavaScript. The project is designed to simulate a real-world productivity and task tracking application where users can organize and manage tasks through an interactive dashboard interface.

This project is currently in development and is being built to strengthen front-end development skills including responsive layouts, DOM manipulation, dynamic rendering, and application-style UI design.

---

## Planned Features

-Create new tasks
-Edit and delete tasks
-Mark tasks as completed
-Filter tasks by status and priority
-Search tasks dynamically
-Responsive dashboard layout
-LocalStorage data persistence
